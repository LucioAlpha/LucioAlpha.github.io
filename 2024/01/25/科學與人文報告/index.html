# ❓ 語音辨識的目標

在 Anaconda jupyter 中使用 Python 寫語音指令。其功能應包括以下幾點：

  1. 它應能重複運作。
  2. 它應該能夠輸出說話者的聲音。
  3. 它應該能夠識別 “zh-TW”。
  4. 它應該能夠在說話者呼吸時自動換行或調整句子。
  5. 當偵測到說話者沒有說話後，一定時間後應清除所有輸出的句子。

import speech_recognition

因為是在 Jupyter 上執行，語音輸入與文字輸出是在兩個不同的 cell 中執行的，故將程式分開擺放。

# 多次語音輸入辨識測試

參考 Chatgpt 提供的程式

import speech_recognition as sr
import pyttsx3
import time

# Initialize the recognizer and text-to-speech engines
recognizer = sr.Recognizer()
text_to_speech = pyttsx3.init()

# Function to clear the output cell
def clear_output():
    from IPython.display import clear_output as clear
    clear(wait=True)

while True:
    with sr.Microphone() as source:
        print("Listening... Say something:")
        audio = recognizer.listen(source)

    try:
        # Recognize speech using Google Web Speech API
        text = recognizer.recognize_google(audio, language = 'zh-TW')

        # Output the speaker's voice
        print(f"You said: {text}")

        # Split text into sentences and output with line breaks
        sentences = text.split('.')
        for sentence in sentences:
            text_to_speech.say(sentence)
            text_to_speech.runAndWait()

    except sr.UnknownValueError:
        print("Could not understand audio.")

    except sr.RequestError as e:
        print(f"Could not request results; {e}")

    # Add a delay to clear output sentences after a certain period
    time.sleep(60)  # Adjust the delay in seconds as needed to clear output

    # Clear the output cell
    clear_output()

# 最終版本

我們在多次測試語音辨識之後發現了一個問題，就是:

「語音辨識會一直偵測說話者的語音直到他停下來」

這將會導致無法達成即時翻譯，而是造成如同 RPG 對話般，你一句我一句的回合式對話。

這是我們最不想見到的事情。

audio = recognizer.listen(source, 6, 4)

audio = recognizer.listen (音訊來源，聽的持續時間，在聽完之後輸出的間隔時間)

這一條敘述句就是我們的最佳解答，透過限制每一段偵測的時間，與輸出間隔時間，我們可以掌握字幕顯示的字數與長短，進而解決目前所遇到的問題。

# 目前辨識的最終版本
import speech_recognition as sr
import pyttsx3
import time

# Initialize the recognizer and text-to-speech engines
recognizer = sr.Recognizer()
text_to_speech = pyttsx3.init()

# Function to speak text
def speak_text(text):
    text_to_speech.say(text)
    text_to_speech.runAndWait()

# Main loop
while True:
    with sr.Microphone() as source:
        print("Listening... Say something:")

        # audio = recognizer.listen(音訊來源，聽的持續時間，在聽完之後輸出)
        # 合格的數值 1. 6,4  2. 5 4
        audio = recognizer.listen(source, 4, 4) 

    try:
        # Recognize speech using the 'zh-TW' (Chinese - Taiwan) language model
        text = recognizer.recognize_google(audio, language="zh-TW")

        print(f"You said: {text}")

        # # Speak the recognized text in real-time
        # speak_text(f"You said: {text}")



    except sr.UnknownValueError:
        print("Could not understand audio.")

    except sr.RequestError as e:
        print(f"Could not request results; {e}")

    # Output the result every 2 seconds
    #time.sleep(2)

    # Clear the console every 15 seconds
    if time.time() % 15 == 0:
        print("\033c", end="")  # Clear the console

# 偵測不到中文

以單次語音輸入辨識測試來舉例，因為其進行辨識時，沒有指定是哪種語言模型。

# 測試用的單次收音辨識
import speech_recognition
r = speech_recognition.Recognizer()

with speech_recognition.Microphone() as source:
    audio = r.listen(source)
    
r.recognize_google(audio)

所以我們應該在第 8 行做更改，指定語言模型:

# 測試用的單次收音辨識
import speech_recognition
r = speech_recognition.Recognizer()

with speech_recognition.Microphone() as source:
    audio = r.listen(source)
    
# 'zh-TW' 或者 'zh-CN'
r.recognize_google(audio, language='zh-TW')
