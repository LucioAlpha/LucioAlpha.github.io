# 中文翻譯

給予衛星到地球表面的距離及衛星間的角度，以求最短的弧長及弦長。

# 程式碼一覽

# C++ 版本

# 解題思路

  1. 先將 arc minute 化為角度，並將角度化為最小，以求最短弧長
    1. （若角度 > 360，則 angle % 360；若角度 > 180，則 angle = 360-angle）
  2. 套用公式：
    1. 弧長 = 2 * 半徑 * PI * (角度 / 360)
    2. 弦距 = sin (角度) * 半徑 * 2
