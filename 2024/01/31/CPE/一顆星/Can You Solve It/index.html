# 對角線編號公式推導 = x + y

Loc(x,y)=前面(含本身)所有對角線元素的總和+xLoc(x, y) = 前面(含本身)所有對角線元素的總和+xLoc(x,y)=前面(含本身)所有對角線元素的總和+x

Loc(x,y)=(0+1+2+3+...+本身對角線編號)+xLoc(x, y) = (0+1+2+3+...+本身對角線編號)+xLoc(x,y)=(0+1+2+3+...+本身對角線編號)+x

Loc(x,y)=(對角線編號)(對角線編號+1)2+xLoc(x, y) = \frac{(對角線編號)(對角線編號+1)}{2}+xLoc(x,y)=2(對角線編號)(對角線編號+1)​+x

Loc(x,y)=(x+y)(x+y+1)2+xLoc(x,y) = \frac{(x+y)(x+y+1)}{2}+xLoc(x,y)=2(x+y)(x+y+1)​+x
