# 中文翻譯

假設一座標軸上有一個平行四邊形

給定平行四邊形的兩個相鄰邊的端點的 (x, y) 座標，找到第四個點的 (x, y) 座標。

# 輸入

輸入包含多組測試資料。

每組測試資料含有 8 個浮點數。

  • 第一個、第二個浮點數代表第一個邊的端點 (x, y) 座標
  • 第三個、第四個浮點數代表第一個邊的另一端點 (x, y) 座標
  • 第五個、第六個浮點數代表第二個邊的端點 (x, y) 座標
  • 第七個、第八個浮點數代表第二個邊的另一端點 (x, y) 座標

所有座標均以公尺為單位，請精確到毫米單位 (小數點後 3 位)。

所有座標均在−10000 和 + 10000 之間。

# 輸出

對於每行輸入

請輸出平行四邊形的第四個點的 (x, y) 座標

以公尺為單位，精確到毫米 (小數點後 3 位)

(x, y) 座標請用一個空格隔開。

# 解題思路

平行四邊形的中心點公式：

(A+C)2=(B+D)2\frac{(A+C)}{2} = \frac{(B+D)}{2}2(A+C)​=2(B+D)​

所以假設第四點是 D ，那公式可以整理成

D=A+C−BD = A+C-BD=A+C−B

因為輸入的是兩個相鄰邊的點，共四個座標，所以或有一點重複

假設重複的是Ｂ，Ａ與Ｃ沒有重複，座標點總和Ｓ如下：

S=A+C+2BS = A+C+2BS=A+C+2B

而當我們找到重複的 B 的之後，將總和減去 B 的 3 倍，即可找出 D （第四點座標）

S−3B=A+C−B=DS-3B = A+C-B = DS−3B=A+C−B=D
