# 中文翻譯

題目開頭在推導微分公式，此題要求根據題目的輸入值，求出導數（微分過後的方程式）的值。

f(x)=a0xn+a1xn−1f(x) = a_{0}x^n+a_{1}x^{n-1}f(x)=a0​xn+a1​xn−1

f′(x)=(a0×n)xn−1+(a0×(n−1))xn−2f'(x) = (a_{0} \times n)x^{n-1} + (a_{0} \times (n-1))x^{n-2}f′(x)=(a0​×n)xn−1+(a0​×(n−1))xn−2

  • 輸入的第一列會告訴 X 的值是多少
  • 輸入的第二列會以降冪的方式輸每個 X 的係數
  • 最高次方會是 1000000

舉例來說，X = 7，第二列為 2 1 3 4：

7
2 1 3 4

可以得出，還未微分的原方程式為：

2x3+x2+3x+42x^3+x^2+3x+42x3+x2+3x+4

根據題目所給的微分公式：係數 * 次方、次方 - 1、原常數項消除

6x2+2x+36x^{2}+2x+36x2+2x+3

此時代入題目所給的 X = 7

6×72+2×7+3=3116 \times 7^{2}+2 \times 7+3 = 3116×72+2×7+3=311

得到最終答案 311

# 解題思路

  1. 處理輸入值，注意輸入之兩個係數間可能含多個空白、不知係數數量未知

  2. 使用 long long int 型別，依算式計算出微分答案 (注意 x = 0 之結果)

# 如何實現微分？

使用整數陣列存取係數，索引值當作（次方－總數）
