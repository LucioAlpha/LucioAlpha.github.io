You can say 11

星星難度: ⭐
類型：基礎題
題號: UVA10929

# 題目一覽（照片或 PDF 檔）

p10929.pdf

# 中文翻譯

給定一個數字，是求它是否為 11 的倍數

當輸入的數字為 0 時，程式結束

數字大小：1 ~ 1000 位數

# 程式碼一覽

# C++ 版本

# 解題思路

想到要解跟倍數有關的問題時，就要想 **【％】取餘數 ** 的功能

11 的倍數判別法為「奇數位數字和」減 「偶數位數字和」等於 0 或 11 的倍數

又因為題目的數字是 1~1000 位數的數字，正常的 int 和 long long int 都無法裝下

所以要使用字串【string】或者字元陣列【char []】來存取題目的輸入
