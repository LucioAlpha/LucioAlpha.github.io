# 中文翻譯

給定一個範圍（a、b），請找出範圍內的完全平方數（可以被開根號出整數）

  • 範圍的限制：0 < a ≤ b ≤ 100000

# 解題思路

# 正常版本

  1. 檢查開根號後取整數的值，反乘回去後是否等同於原本的值
    1. 如果是，就是完全平方數，計數器 + 1
    2. 如果不是，那不是完全平方數

# 精簡版本

  1. 計算 a 及 b 之平方數，並相減算出其個數
    1. 需考慮 a 本身是否為完全平方數，所以用 ceil () 計算

# celi ()

回傳大於等於所給數字的最小整數，例如：

celi ( 4.9 ) = 5

# sqrt ()

回傳所給數字開根號後的結果，例如：

sqrt ( 4 ) = 2
