# 中文翻譯

在這個問題中你必須將數列文字往順時針方向旋轉 90 度。也就是說將原本由左到右，由上到下的句子輸出成由上到下，由右到左。

輸入最多不會超過 100 列，每列最多不會超過 100 個字元。

合法的字元包括：換行，空白，所有的標點符號，數字，以及大小寫字母。

（注意：Tabs 並不算是合法字元。）

最後一列輸入必須垂直輸出在最左邊一行，輸入的第一列必須垂直輸出在最右邊一行。

請參考題目 PDF 的 sample intput/output。

# 解題思路

  1. 創建字串陣列或者字元的二維陣列
  2. 將所有輸入值存入 string []，並找出最長字串長度
    1. 注意：宣告陣列大小時要超過 100，否則會 runtime error
  3. 依順序改為直行輸出，記得補足空白
