銘傳資傳系 CPE 042 UVa299-Train Swapping

中文翻譯

  • 輸入火車的排列狀態，輸出需要交換多少次以排好順序
  • 需要注意的有：
    1. 輸入第一行一個整數 N 代表測資數
    2. 每筆測資第一行 L 代表火車長度 (0 ≤ L ≤ 50)
    3. 每次交換只能對調相鄰兩車廂
    4. 輸出格式
      • Optimal train swapping takes S swaps.

程式碼一覽

C++版本（泡沫排序法，Bubble Sort）

C++（合併排序法，Merge Sort）

解題思路

使用泡沫排序法（Bubble sort）進行排序

  1. 用迴圈比較兩兩相鄰元素，若前者較大則交換，並將交換次數++
  2. 最後輸出總交換次數
