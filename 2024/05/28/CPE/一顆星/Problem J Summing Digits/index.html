Problem J Summing Digits

:::info
星星難度: ⭐
類型: 基礎題
題號: UVA11332
:::

題目一覽（照片或PDF檔）

p10222.pdf

中文翻譯

定義一個函式 f(n) 是所有十進位數字位數的總和，再 f(f(n))、f(f(f(n)))……，直到只剩下個位數為止並定義該值為 g(n)。

  • 輸入的每一行會有一個正整數 n，其值最大到 2×10^9
  • 必須輸出g(n)。輸入是以0值做為結束，該值不需要輸出

程式碼一覽

C++版本（語法改一改，Ｃ版本是一模一樣的）

解題思路

用遞迴+迴圈即可解決這一題，用【%】取個位數的數字，【/=】將位數向下推移一位
